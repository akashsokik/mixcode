# Welcome to MixCode

MixCode is a powerful terminal user interface (TUI) that lets you run **Claude Code** and **Codex** side-by-side directly from your terminal. It's built on [OpenTUI](https://opentui.com/) and provides a seamless environment for comparing AI agents, testing approaches, and building with multiple code generation models simultaneously.

## What is MixCode?

MixCode is a custom coding-agent TUI that bridges your terminal with two state-of-the-art AI coding assistants:

- **Claude Code** - Powered by Anthropic's Claude models via the Claude Agent SDK
- **Codex** - Powered by OpenAI's advanced code models

Both agents run in a unified interface with:
- Real-time streaming of model outputs
- Tool execution and logging
- Session management and history
- Project-scoped environments
- Per-directory configuration

## How It Works

```
┌─────────────────────────────────────┐
│         MixCode TUI (React)          │
│    Terminal UI via @opentui/react   │
└──────────────────┬──────────────────┘
                   │ WebSocket (ws)
                   │ JSON Events
┌──────────────────▼──────────────────┐
│    Node.js Backend (Hono Server)    │
│  Manages Sessions & Agent Runners   │
├──────────────┬──────────────────────┤
│              │                      │
│         Claude SDK          OpenAI Codex SDK
│  Agent Orchestration    Agent Orchestration
│                                     │
└──────────────┬──────────────────────┘
               │
          Tool Execution
```

1. **You type** a prompt in the MixCode terminal
2. **TUI sends** the message to the backend via WebSocket
3. **Backend selects** the active runner (Claude or Codex)
4. **SDK processes** the request with tool execution
5. **Events stream** back to the TUI in real-time
6. **You see** the results as they happen

## Quick Start

Get started in seconds:

```bash
# Install and setup
npm install
cp .env.example .env
# Add your API keys to .env

# Start MixCode
npm start

# Or use the global mixcode command (after npm link)
cd ~/your-project
mixcode
```

Then in the TUI:
- Type your prompt and press Enter
- Use `/claude` or `/codex` to switch agents
- Browse with `j`/`k`, create new sessions with `n`
- Press `Esc` to enter browse mode

## Architecture Overview

MixCode consists of three main components:

| Component | Technology | Purpose |
|-----------|-----------|---------|
| **TUI** | Bun + React + @opentui/react | Terminal UI rendering and user interaction |
| **Server** | Node.js + Hono + WebSocket | Session management and agent orchestration |
| **Shared** | TypeScript | Protocol definitions and type sharing |

Each runs independently but communicates over a local WebSocket connection. The server maintains session state and SDK continuity, while the TUI is a thin stateless projection of that state.

## Requirements

- **Node.js** >= 20
- **Bun** (install with `curl -fsSL https://bun.sh/install | bash`)
- **Anthropic API Key** (for Claude) or **OpenAI API Key** (for Codex)
- Unix-like terminal (macOS, Linux)
