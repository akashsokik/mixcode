# TUI Development

Guide to developing the MixCode terminal user interface.

```

├── src/
│   ├── index.tsx            # React entry point

│   │   ├── useSession.ts
```

## Development

### Start TUI Only

```bash
npm run tui
```

Auto-rebuilds on changes (Bun's hot reload).

### Testing UI Changes

Run backend separately:

```bash
# Terminal 1: Backend
npm run server

# Terminal 2: TUI
npm run tui
```

Edit components, refresh UI.

## Architecture

### State Management

MixCode uses React hooks for state:

```typescript
// useSession.ts - Session management
const [session, setSession] = useState<Session | null>(null)
const [runner, setRunner] = useState<RunnerKind>("claude")

// useHistory.ts - Message history
const [messages, setMessages] = useState<Message[]>([])

// useCompletion.ts - Streaming completion
const [delta, setDelta] = useState("")
```

### WebSocket Connection

```typescript
// api/ws.ts
class WebSocketClient {
  connect(url: string): Promise<void>
  send(msg: ClientMsg): void
  on(type: string, handler: (msg: any) => void): void
  close(): void
}
```

## Common Tasks

### Add Component

1. Create `/tui/src/components/MyComponent.tsx`
2. Use React functional component
3. Import in `app.tsx`
4. Add to layout

### Add Hook

1. Create `/tui/src/state/useMyHook.ts`
2. Export hook function
3. Use in components: `const [state, setState] = useMyHook()`

### Add Utility

1. Create `/tui/src/util/myutil.ts`
2. Export functions
3. Import where needed: `import { myFunc } from "../util/myutil"`

## Styling

MixCode uses terminal-native styling via `@opentui/react`:

```typescript
<Text color="cyan">Styled text</Text>
<Box padding={1} borderStyle="round">
  Content
</Box>
```

No CSS needed - all terminal-native.

## Testing

### Unit Tests

```typescript

import { describe, it, expect } from "vitest"
import { myFunction } from "../myfile"

describe("myFunction", () => {
  it("works correctly", () => {
    expect(myFunction("test")).toBe("expected")
  })
})
```

### Run Tests

```bash
npm run test
npm run test:watch
```
